Courses

Course descriptions are taken from the Cornell class roster.

Most of these courses have names beginning with "ORIE", but some do not. ENGRI 1101 is an engineering introduction course, so it must have "ENGRI" as its designation, but it is definitely an ORIE course. ENGRD 2700, while not an "ORIE" course, is a course that is taught by professors in the ORIE department and is taken by all ORIE students.

Note: only a small subset of ORIE courses are listed here. To add more courses, please make a pull request.

Note: some links listed here may be broken. To fix this, please create pages corresponding to those courses by making a pull request.
