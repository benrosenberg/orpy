ORIE 4820

Course description (from class roster):

Students develop and implement practical spreadsheet models to analyze data and evaluate decision problems in a hands-on learning environment. Microsoft Excel is heavily used. A wide variety of application areas are covered that incorporate concepts from probability, statistics, and constrained optimization.

Offered: Spring.

Prerequisites: ENGRD 2700, ORIE 3300, or equivalent.

Is Python used?

No. This course is taught mainly in Excel and VBA. You don't really need to pre-study anything for it either, as whatever VBA is used will be taught in class. It might be nice to have some Excel experience going in.
